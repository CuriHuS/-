## 게임 플레이(Game Play)
> 💡 How to play this game?

### 조작법
**이동(Move)**: `W,A,S,D` <br>

**점프(Jump)**: `Space bar`<br>

**밀기(Push)**: `E`<br>

**매달리기(Hang)** : `F`<br>

**리스폰(Respawn)**: `R`

**시점 변환(Change View)**: `V`
