### 조작법(Control)
**이동(Move)**: `W,A,S,D` <br>

**달리기(Sprint)** : `Left Shift` <br>

**점프(Jump)**: `Space bar`<br>

**밀기(Push)**: `E`<br>

**매달리기(Hang)** : `F`<br>

**시점 변환(Change View)**: `V`


### 클리어 조건(How to clear)
> MSG를 모아 목표 요리사에게 도달하라!

- 고양이나 요리사에게 들키면 쫓아옵니다
- 맵 곳곳에 요소들을 이용해 따돌리세요
